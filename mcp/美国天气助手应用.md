# 安装 UV 及 创建虚拟环境 以及 编写天气助手应用的代码 
```powershell
uv init weather_test
cd .\weather_test\
uv venv
# source .\.venv\Scripts\activate
.\.venv\Scripts\activate

# touch weather_test.py
New-Item -ItemType File -Name "weather_test.py"
```
代码详情见 [weather_test.py](weather_test/weather_test.py)

# 配置 MCP 服务器
```json
{
  "mcpServers": {
    "weather": {
      "command": "uv",
      "args": [
        "--directory",
        "D:\\code repository\\my_code_repo\\ai_study\\mcp\\weather_test",
        "run",
        "weather_test.py"
      ]
    }
  }
}
```
